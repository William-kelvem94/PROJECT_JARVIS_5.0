# JARVIS Test Files

Esta pasta contém arquivos de teste e debug para o JARVIS 5.0.

## Arquivos Recentes (Fevereiro 2026)

### Scripts de Teste
- `test_interface.py` - Testa se a interface PyQt6 está funcionando
- `test_jarvis_startup.py` - Testa inicialização completa do JARVIS

### Arquivos de Debug/Output
- `debug_output.txt` - Output de debug da última execução do JARVIS
- `jarvis_output.txt` - Logs de saída do JARVIS

### Como Usar
```bash
# Testar interface
python tests/test_interface.py

# Testar inicialização completa
python tests/test_jarvis_startup.py
```
