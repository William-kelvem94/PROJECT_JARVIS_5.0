# 🗄️ JARVIS 5.0 - Archive

Esta pasta contém arquivos, scripts e logs antigos arquivados para referência histórica.

⚠️ **ATENÇÃO:**
- **NÃO** execute scripts desta pasta diretamente. Eles podem conter código depreciado, incompatível ou que quebre a instalação atual.
- **NÃO** comite arquivos grandes ou sensíveis aqui.
- O conteúdo desta pasta é **ignorado** pelo Git (exceto este README).

## Conteúdo Arquivado

- `gui_old/`: Versões antigas da interface gráfica.
- `jarvis_core/`: Versões anteriores do núcleo do sistema.
- `archived_logs/`: Logs de sessões passadas.
- `prototypes/`: Protótipos experimentais descartados.

Mantenha esta pasta limpa para evitar confusão.
