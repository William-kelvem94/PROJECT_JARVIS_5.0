## Overview

A Next.js frontend for a simple AI voice assistant interface. This starter provides UI components for voice interaction, chat transcripts, visual activity indicators, and optional camera preview.

## Deployment

This app is designed to run locally with a standard Next.js setup and does not depend on an external sandbox service for the core frontend.

## Installation

To run this application locally, clone the repo and install dependencies:

```console
git clone <repo-url>
cd frontend
pnpm install
pnpm dev
```

For more information about the frontend, refer to the README in this folder.
