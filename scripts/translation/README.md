# 🔤 Scripts de Tradução

Esta pasta contém scripts utilitários para tradução e localização do projeto JARVIS.

## 📋 Arquivos Disponíveis

- **`git_msg_filter.py`** - Filtro para tradução de mensagens do Git
- **`msg_filter.py`** - Utilitário geral de filtragem de mensagens
- **`show_translations.py`** - Visualizador de traduções disponíveis
- **`translate_commits_automated.py`** - Sistema automatizado de tradução de commits
- **`commit_translations_map.txt`** - Mapeamento de traduções de commits

## 🚀 Uso

```bash
# Visualizar traduções disponíveis
python scripts/translation/show_translations.py

# Executar tradução automatizada de commits
python scripts/translation/translate_commits_automated.py
```

## 📖 Sobre

Estes scripts foram desenvolvidos para:
- **Automatizar traduções** de commits e mensagens
- **Manter consistência** na localização
- **Facilitar colaboração** em múltiplos idiomas
- **Documentar mudanças** em português

**Nota**: Os scripts de tradução são ferramentas de desenvolvimento e podem requerer configuração específica.
