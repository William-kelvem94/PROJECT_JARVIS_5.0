# Research Archive

This directory stores historical research, prototypes, and future-facing ideas that are not part of
the active JARVIS runtime.

## Contents

- `FUTURE_IMPLEMENTATIONS/` - legacy or deferred implementation sketches.
- `OMEGA_RESEARCH/` - older research notes and prototypes.
- `PROJECT_SOPHISTICATION/` - design notes, development logs, and sophistication proposals.

Keep active runtime code in `backend/`, `frontend/`, `scripts/`, `src/`, or `tools/`. Promote files
from this archive only after they are reviewed, tested, and wired into the current project structure.
