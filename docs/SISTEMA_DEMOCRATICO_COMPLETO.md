
**Como funciona:**

## 🌐 REDE DEMOCRÁTICA INTELIGENTE

### Como Funciona a Democracia
1. **Descoberta**: Cada PC escaneia a rede local
2. **Identificação**: PCs se identificam via Microsoft Account
3. **Eleição**: PCs votam democraticamente em um líder  
4. **Distribuição**: Tarefas são distribuídas por capacidade
5. **Rotatividade**: Líder pode mudar automaticamente

### Cenários de Uso

**📱 Exemplo 1: Conectar Laptop**
```
1. Laptop entra na rede WiFi
2. JARVIS detecta automaticamente
3. Laptop se identifica (williamkelvem64@gmail.com + fingerprint)
4. Sistema pergunta se quer incluir na rede democrática
5. Você autoriza pela interface
6. Laptop participa das eleições
```

**🖥️ Exemplo 2: PC Principal como Líder**
```
1. PC principal tem melhor GPU → se candidata a líder
2. Outros PCs votam baseado nas capacidades
3. PC principal vira líder democrático
4. Controla treinamento de IA distribuído
5. Outros PCs contribuem com CPU/dados
```

**⚡ Exemplo 3: Failover Automático**
```
1. PC líder desliga inesperadamente
2. Sistema detecta líder offline
3. Força nova eleição democrática
4. Próximo PC mais capaz assume liderança
5. Operações continuam automaticamente
```

**Via Interface:**

2. **Selecionar dataset**: Seus dados ou gerar automaticamente


## 🚨 TROUBLESHOOTING

### Problemas Comuns

**❌ "Microsoft Account não detectada"**
```
Solução: Interface → Identidade → ⚙️ Configurar Manualmente
Inserir: williamkelvem64@gmail.com
```

**❌ "Google Drive não encontrado"**
```
Solução: Interface → Identidade → 🔗 Conectar Google Drive
Verificar se Google Drive está instalado e rodando
```

**❌ "Nenhum dispositivo na rede"**
```
Solução: Interface → Dispositivos → 🔍 Escanear Rede
Verificar se PCs estão na mesma rede WiFi/LAN
```

**❌ "Eleição democrática falhou"**  
```
Solução: Interface → Rede Democrática → ⚡ Força Democrática
Fazer reset completo do sistema democrático
```

**❌ "Verificação biométrica falha"**
```
Solução: Interface → Identidade → 👤 Configurar Perfil
Reconfigurar amostras de face/voz
```
