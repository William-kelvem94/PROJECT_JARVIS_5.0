## 📋 Convenções da Documentação

### Formatação
- **Títulos:** Hierarquia clara com ##, ###, ####
- **Código:** Blocos com ```python ou ```yaml
- **Ícones:** Uso consistente de emojis por categoria
- **Links:** Relativos dentro de docs/

### Nomenclatura
- **Arquivos:** kebab-case (ex: `learning-system.md`)
- **Pastas:** kebab-case (ex: `getting-started/`)
- **Títulos:** Title Case com emoji (ex: `## 🚀 Instalação`)

### Status dos Documentos
- ✅ **Completo:** Documento finalizado e revisado
- 🚧 **Em Progresso:** Documento em construção
- 📝 **Rascunho:** Notas preliminares
- ⚠️ **Deprecado:** Não usar, ver versão nova
