# Questionário — Quem é Will?

> Este é o questionário de onboarding que o Jarvis usa para conhecer Will.
> As respostas alimentam `Will-Pessoal/Perfil/Perfil.md`, `Will-Pessoal/Perfil/Preferencias.md`, `Will-Pessoal/Objetivos/README.md` e `Will-Pessoal/Vida/Rotina.md`.
> Marcar [x] quando a pergunta for respondida e a resposta já estiver no arquivo correto.

## 👤 Identidade e Vida

- [ ] Qual é o seu nome completo e como prefere ser chamado?
- [ ] Em que cidade/estado você mora?
- [ ] Qual é a sua idade?
- [ ] Você trabalha, estuda, ou os dois?
- [ ] Tem uma empresa/freelance ou é CLT?
- [ ] Com quem você mora?
- [ ] Quais são seus maiores interesses fora do trabalho?

## 💻 Trabalho e Tecnologia

- [x] Qual é a sua stack principal? → *Python, TS, PHP - ver Perfil.md*
- [x] Quais projetos estão ativos agora? → *ver Projetos-Ativos.md*
- [ ] Qual é o seu projeto mais importante hoje?
- [ ] Qual é a sua maior frustração técnica recente?
- [ ] Quando você programa mais: manhã, tarde ou noite?
- [ ] Você tem sócios ou trabalha sozinho?
- [ ] Já tem clientes ou está em fase de construção?

## 🎯 Objetivos e Sonhos

- [x] Quais são seus OKRs para 2026? → *ver Objetivos.md*
- [ ] Qual é o sonho grande de 5 anos?
- [ ] O que te faria sentir que este ano foi um sucesso?
- [ ] O que você mais quer aprender este ano?
- [ ] Tem algum objetivo financeiro específico?

## ⚡ Energia e Produtividade

- [ ] Qual é o seu maior inimigo de produtividade?
- [ ] Como você sabe quando está no "flow"?
- [ ] O que te desanima mais no trabalho?
- [ ] Qual é o seu ritual para começar a trabalhar?
- [ ] Como você prefere que o Jarvis te ajude: direto, detalhado, ou conforme o contexto?

## 🤖 Sobre o Jarvis

- [ ] O que você espera do Jarvis no dia a dia?
- [ ] Quais tarefas você mais quer delegar para o Jarvis?
- [ ] Tem alguma coisa que o Jarvis NUNCA deve fazer?
- [ ] Como prefere que o Jarvis lide com erros (dele ou seus)?
- [ ] Você quer que o Jarvis seja mais formal ou mais parceiro?

## 🎵 Pessoal

- [ ] Qual é a sua música/gênero favorito?
- [ ] Que tipo de conteúdo você consome (YouTube, podcasts, livros)?
- [ ] Tem algum hábito seu que o Jarvis deve saber para respeitar?
- [ ] Tem algum assunto pessoal que prefere não discutir com o Jarvis?

---
*Este arquivo deve ser respondido gradualmente. O Jarvis pode fazer uma pergunta por vez durante interações naturais.*
