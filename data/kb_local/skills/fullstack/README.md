# Fullstack Skills

Esta pasta reúne skills e prompts para desenvolvimento fullstack com foco em IA, automação e projetos práticos.

## Estrutura
- [[INDEX|Índice completo do hub]] — índice de notas do Fullstack.
- [[frontend|Frontend]] — prompts e MCP para React, Vue e interfaces.
- [[backend|Backend]] — prompts e MCP para APIs, serviços e Python/Node.
- [[database|Database]] — prompts e MCP para bancos SQL/NoSQL e migrações.

## Como usar
1. Abra o arquivo que corresponde ao domínio em que está trabalhando.
2. Use os prompts como ponto de partida para gerar código ou revisar implementações.
3. Use as ferramentas MCP para localizar arquivos, editar trechos e validar com comandos.
4. Adapte os exemplos ao contexto do projeto `PROJECT_JARVIS_5.0` ou outros projetos no vault.

## Dicas
- Combine `skills/vscode-ai` com `skills/fullstack` para abordagens mais completas.
- Use `skills/fullstack/backend.md` quando alterar APIs e serviços.
- Use `skills/fullstack/database.md` para modelagem de dados e consultas.
- Use `skills/fullstack/frontend.md` para componentes, UX e interfaces.
